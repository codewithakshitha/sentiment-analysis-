# Twitter Sentiment Analysis Dashboard

This project analyzes Twitter tweets and classifies sentiments as Positive, Negative, or Neutral. It also visualizes sentiment insights using an interactive dashboard.

## Features

* Sentiment Classification
* Tweet Score Analysis
* Topic-wise Sentiment Analysis
* Polarity vs Subjectivity Graph
* Sentiment Trends Over Time
* Dashboard Visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* NLP

## Files

* twitter_sentiment_analysis.py
* sentiment_dashboard.png
* sentiment_results.csv
